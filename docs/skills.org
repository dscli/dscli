* Skill 基本结构
  参照 [[../internal/skills][internal/skills]] 以及以下例子了解Skill的基本结构。
  #+begin_src text
    $ tree ~/.dscli/skills
    /home/nanjj/.dscli/skills
    ├── skill-name               # Skill name
    │   └── SKILL.md             # Skill markdown file 
    └── skills.yaml              # dumped skills yaml after load

    2 directories, 2 files

    $ cat ~/.dscli/skills/skill-name/SKILL.md

    ---
    name: skill-name                # Skill name
    description: 这是一个全局测试技能，用于验证list命令功能。 # skill description
    keywords: [global, test, skill]                           # trigger on keywords
    ---

    这是一个全局测试技能，用于验证list命令功能。

    ## 内容

    ```bash
    echo "这是一个全局测试技能"
    echo "用于验证dscli skill list命令功能"
    ```
  #+end_src
