## 技术细节

### 核心组件
1. **syntax.Parser**：解析 Shell 脚本
2. **interp.Runner**：执行解析后的 AST
3. **expand.Environment**：环境变量管理

### 关键配置选项
```go
interp.StdIO(stdin, stdout, stderr)  // IO 重定向
interp.Env(env)                      // 环境变量
interp.Dir(dir)                      // 工作目录
interp.Params("-e", "-u")           // Shell 参数
```

### 安全处理器
1. **ExecHandler**：控制命令执行
2. **OpenHandler**：控制文件访问
3. **ReadDirHandler**：控制目录读取
