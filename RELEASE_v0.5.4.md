# dscli v0.5.4 发布说明

## 🎉 版本亮点

**v0.5.4** 版本引入了实用的余额显示功能，让用户在使用AI对话时能随时了解账户余额情况，避免余额不足的尴尬。

## ✨ 新功能

### 💳 余额显示和低余额提醒
- **会话结束后显示余额**：每次AI对话结束后，除了显示会话花费，还会显示当前余额
- **低余额提醒**：当余额低于10元时，会显示醒目的提醒信息，提示用户及时充值
- **直观的图标**：
  - ⏱️ 会话用时
  - 💰 会话花费  
  - 💳 当前余额
  - ⚠️ 余额较低提醒

### 示例输出
```
优化前（三行）：
⏱️  会话用时: 30.0s
💰  会话花费: CNY 4.50
💳  当前余额: CNY 95.50

优化后（一行）：
⏱️ 30.0s  💰 CNY 4.50  💳 CNY 95.50

当余额较低时：
⏱️ 30.0s  💰 CNY 95.00  💳 CNY 5.00  ⚠️ 余额较低，请及时充值！
```

## 📦 安装方式

### 方式1：使用 go install（推荐）
```bash
go install gitcode.com/dscli/dscli@v0.5.4
```

### 方式2：从源码构建
```bash
git clone https://gitcode.com/dscli/dscli.git
cd dscli
git checkout v0.5.4
make install    # 安装到 $GOPATH/bin
```

### 方式3：下载预编译二进制
查看 [Releases 页面](https://gitcode.com/dscli/dscli/releases) 获取最新版本
